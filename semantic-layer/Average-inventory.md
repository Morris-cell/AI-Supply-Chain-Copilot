# Average Inventory

## Definition

Average inventory value held during a specific period.

Represents the typical inventory level used for inventory efficiency and working capital analysis.

## Formula

Average Inventory = (Beginning Inventory Value + Ending Inventory Value) / 2

## Possible Field Names

avg_inventory
average_inventory
avg_inventory_value
inventory_avg
average_stock_value

## Data Type

Currency

## Related KPIs

- Inventory Turnover
- Days Inventory Outstanding (DIO)
- Inventory Carrying Cost
- Cash Conversion Cycle (CCC)
- Inventory-to-Sales Ratio

## Notes

Average Inventory is one of the most fundamental inventory management concepts.

It is commonly used as the denominator in Inventory Turnover calculations and the numerator in DIO calculations.

Inventory Turnover = COGS / Average Inventory

DIO = (Average Inventory / COGS) × Number of Days
